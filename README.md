# 6-Automated-Cost-Optimizer-using-AWS
AWS automation project that identifies underutilized resources and helps optimize cloud infrastructure costs efficiently.

This repository presents an automated AWS cost optimization solution designed to monitor and reduce unnecessary cloud expenses. The project uses AWS services and automation scripts to identify underutilized resources, generate reports, and implement optimization strategies. It demonstrates practical cloud cost management concepts that help organizations improve resource efficiency and minimize operational expenses. The project is valuable for learning cloud governance, monitoring, and financial optimization practices.

AWS Services Used :

                  AWS Lambda 
                  Amazon EC2 
                  Amazon CloudWatch
